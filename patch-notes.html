Patch Notes History

All versions starting at v1.0 will have their notes displayed here. Always updated when a new patch is released. The most recent patch notes will be at the top of the page

v1.1 - Powers

2/3/2020

    Main Changes
  • High Score tracker added
  • 3 power-ups added (rapid fire, player speed boost, bigger enemies)
  • +1 life after every 5 levels
  • Levels 11-15 added
  • New enemy type: attack - will shoot at player
    Bug Fixes
  • Player can no longer rapid fire unless they have the power-up

v1.0.1 - Difficulty Patch

1/27/2020

    Main Changes
  • Level difficulties changed to make higher levels reachable without cheesing mechanics

v1.0 - First Release

1/27/2020

    Main Changes
  • Smoothed out controls
  • Pause screen added
  • 10 levels created
    Bug Fixes
  • Collision with fast-moving enemies works correctly
  • Key presses (including shooting) no longer interrupt movement
